# ClassView 答題卡 📚

> 為課堂答題而生 — 簡單、清晰、即時

**🏫 匡智張玉瓊晨輝學校**

類似 [Plickers](https://get.plickers.com/) 但**更簡單**：每個學生有 **6 張紙卡**（A / B / C / D / 剔 / 交叉），每張卡 **正面 = QR code**、**背面 = 學生名 + 答案類型**，列印後**對摺**就變兩面卡。學生舉起揀嘅卡，老師用鏡頭掃描自動記錄。

## ✨ 特色

- **📝 兩種題型** — 4 選 1（A/B/C/D）和 2 選 1（✓/✗ 是非題）
- **🃏 6 張紙卡 / 學生** — 每個答案獨立一張卡（A / B / C / D / 剔 / 交叉）
- **📷 自動掃描** — 鏡頭讀到 QR 即記錄（編碼：`P:學生ID:答案`），無需確認
- **📄 PDF 匯出** — 一鍵生成可列印嘅 PDF 紙卡
- **🖨️ 列印友好** — A4 紙 3×4 排版（12 張卡 = 2 學生），摺痕虛線
- **👨‍🎓 學生管理** — 自動產生專屬 QR code，可批次加入
- **🖱️ 手動模式** — 不方便用鏡頭時可逐位點選
- **🔊 語音朗讀** — 內建 TTS 廣東話讀題，照顧讀寫障礙學生
- **👁️ 揭曉答案** — 派完卡即場顯示正確選項
- **📊 即時結果** — 圖表分析全班及每題正確率
- **📥 CSV 匯出** — 答題結果可匯出做評估紀錄
- **💾 無需後端** — 全部資料 localStorage 儲存，隱私安全

## 🚀 本地執行

只需要一個 HTTP server（檔案系統直接開啟 `index.html` 會因 CORS 問題無法運作）：

```bash
# 用 Python
python -m http.server 8765

# 或用 Node
npx http-server -p 8765
```

然後開 `http://localhost:8765`。

## 🌐 部署到 Render（永久免費）

1. 把這個資料夾推到 GitHub
2. Render 後台 → New → Static Site
3. 連 GitHub repo，自動偵測 `render.yaml`
4. 部署完成，免費 subdomain

> 第一次部署需用 HTTPS 才能用鏡頭（瀏覽器限制）

## 📋 使用流程

1. **建立題目** → 導航「📝 題目」→ 新增 → 揀四選一或二選一 → 填題目和選項 → 標記正確答案
2. **加入學生** → 導航「👨‍🎓 學生」→ 逐個加入或批次貼上
3. **列印紙卡** → 導航「🖨️ 紙卡」→ 「📥 匯出 PDF」或「🖨️ 列印」→ 剪開 + 對摺 → 派給學生
4. **開始答題** → 導航「▶️ 答題」→ 選擇題目 → 學生**舉起**揀嘅卡 → 鏡頭掃描自動記錄
5. **查看結果** → 導航「📊 結果」→ 點「查看詳情」→ 圖表分析

## 🃏 紙卡設計

每張卡 = **整張 A4 紙**（摺痕在中間），列印後對摺變成兩面卡：

```
┌──────────────────────┐
│  📷 正面：QR Code    │
│                      │
│     [大 QR CODE]     │  ← 摺起後其中一面（A4 size）
│                      │
├──────────────────────┤ ← 剪開 + 對摺
│  📝 背面：姓名 + 類型 │
│  ┌──────────────┐    │
│  │   陳小明     │    │
│  │   學號 #1    │    │  ← 摺起後另一面（彩色底）
│  │      A       │    │
│  └──────────────┘    │
└──────────────────────┘
```

**每個學生 6 張卡**（每張獨立一頁 A4）：

| 卡 | 顏色 | 用於 |
|---|---|---|
| **A** | 🔴 紅 | 4 選 1 題 |
| **B** | 🔵 藍 | 4 選 1 題 |
| **C** | 🟢 綠 | 4 選 1 題 |
| **D** | 🟡 黃 | 4 選 1 題 |
| **✓ 剔** | 🟢 綠 | 2 選 1 題 |
| **✗ 交叉** | 🔴 紅 | 2 選 1 題 |

**PDF 匯出**：每張卡獨立一頁 A4
- 1 個學生 = 6 頁
- 20 個學生 = 120 頁 PDF
- 用 **html2canvas 截圖** PDF，完整支援中文
- 用 **jsPDF** 生成可下載 PDF 檔案

**學生使用方式**：
1. 睇老師螢幕上嘅題目
2. 揀答案 → 舉起對應嘅卡（QR 面向老師）
3. 老師用鏡頭掃描 → 自動記錄

**A4 排版（列印模式）**：1 張卡 = 整頁 A4，方便剪同摺

**A4 排版（紙卡預覽）**：3 列 × 4 行 = 12 張細卡預覽，純綷睇下樣

## 🛠️ 技術棧

- 純 HTML + CSS + JavaScript（無 framework）
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) — QR code 生成
- [jsQR](https://github.com/cozmo/jsQR) — 鏡頭掃描
- [jsPDF](https://github.com/parallax/jsPDF) — PDF 生成
- [html2canvas](https://html2canvas.hertzen.com/) — DOM 截圖（解決 PDF 中文字問題）
- [Chart.js](https://www.chartjs.org/) — 結果圖表
- localStorage — 資料儲存

## ♿ 易用設計原則

- 大字體（基礎 17px、按鈕 48-60px 高）
- 高對比配色（深色文字、白色背景、4 色答案區分）
- 圖示輔助文字
- 觸控友善（按鈕最小 44px）
- TTS 語音支援
- 簡化導航（6 個主功能）
- 一頁式操作，無多層選單
- ✓/✗ 符號配合顏色，照顧唔識字嘅學生
